Domanda 1

Risposta non data
Punteggio max.: 5

Testo della domanda

Si implementi la funzione con prototipo

char * get_sel(char * s, char sel);
i cui parametri sono:
  • una stringa s (terminata dal carattere '\0' come di consueto) e
  • un carattere sel diverso da '\0'.
La funzione alloca e restituisce la sotto-stringa di s compresa fra la prima occorrenza (inclusa) di sel e la seconda (esclusa). Se è presente una sola occorrenza di sel, restituisce la sottostringa fino alla fine di s. Se s non contiene alcun carattere sel, restituisce NULL. Se, ad esempio, viene invocato get_sel("Italia Italia",'i') viene ritornata la stringa "a Ital".

Feedback

char * get_sel(char * s, char sel)
{
	char * ret_str=NULL;
	int i;
	
	/* Scorri s fino a quando non termina oppure si trova sel */
	for (; *s != 0 && *s != sel; s++);
	
	if (*s == sel) {
		/* Trovata un'occorrenza. Mi sposto al prossimo char */
		s++;
		/* Alloco spazio fino alla fne di s */
		ret_str = malloc(strlen(s)+1);
		/* Cerco altra occorrenza (se c'e`) */
		for (i=0; s[i] != 0 && s[i] != sel; i++) {
			ret_str[i] = s[i];
		}
		/* Chiudo la stringa */
		ret_str[i] = 0;
		ret_str = realloc(ret_str, strlen(ret_str)+1); /* opzionale, per eventualmente ridurre la memoria occupata */
	}

	return ret_str;
}

Domanda 2

Risposta non data
Punteggio max.: 5

Testo della domanda

Sia data una lista i cui elementi sono definiti dalla seguente struct

struct node {
    int val;
    struct node * next;
};
Si implementi la funzione dal seguente prototipo
struct node * cut(struct node * head, int begin, int end);

dove:

  • head è un puntatore alla testa di una lista;
  • begin e end rappresentano due posizioni all’interno della lista a partire da head, con begin<=end. Le posizioni sono numerare a partire da 0.

La funzione deve restituire, se possibile, la sotto-lista compresa tra le posizioni begin e end deallocando la rimanente parte. La funzione restituisce il puntatore alla testa della lista cosı̀ calcolata o NULL se non è possibile. Ad esempio se la lista puntata da head e` la seguente

head->{4}->{7}->{3}->{8}->{2}->{1}

allora cut(head,2,4) restituisce la lista

{3}->{8}->{2}

e dealloca gli altri nodi. Se end va oltre il termine della lista la funzione deve restituire la sottolista tra begin la fine della lista.

Feedback

/* Versione iterativa */
struct node * cut(struct node * head, int begin, int end)
{
	struct node * p, tmp=NULL;
	int i;

	if (head == NULL)
		/* Lista vuota */
		return NULL;
	for (p=head, i=0; p != NULL && i <= end; i++) {
		/* tmp punta al nodo che precede p. */
		/* Se p punta alla testa, allora tmp e` NULL */
		if (i < begin) {
			/* Tagliare nodo in testa e avanzare head */
			tmp = head;
			head = p = head->next;
			free(tmp);
			tmp = NULL;
		} else {
			tmp = p;
			p = p->next;
		}
	}
	if (tmp != NULL) {
		/* Chiudere ultimo elemento della lista */
		tmp->next = NULL;
	} else {
		/* Se tmp == NULL, non abbiamo salvato alcun nodo */
		head = NULL;
	}

	/* Tagliare tutto da p in poi */
	while (p != NULL) {
		tmp = p->next;
		free(p);
		p = tmp;
	}

	return head;
}


/* Versione ricorsiva */
struct node * cut(struct node * head, int begin, int end)
{
	struct node * p;
	
	if (head == NULL)
		/* Lista vuota */
		return NULL;
	if (begin > end || end < 0) {
		/* Tagliare tutto da head in poi */
		do {
			p = head->next;
			free(head);
			head = p;
		} while (head != NULL);
		return NULL;
	}
	/* Adesso certamente: begin <= end && end >=0 */
	if (begin > 0) {
		/* Mi sposto in avanti e decremento sia begin che end */
		p = cut(head->next, begin-1, end-1);
		free(head); /* Testa da tagliare */
		return p;
	}
	/* Adesso certamente end >= begin >= 0 */
	head->next = cut(head->next, begin, end-1);
	return head;
}
