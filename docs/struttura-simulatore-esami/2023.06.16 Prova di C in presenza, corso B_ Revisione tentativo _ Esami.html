Domanda 1

Risposta non data
Punteggio max.: 5

Testo della domanda

Data una lista L di valori numerici interi:

  • sia M la sottolista di L che inizia dal suo valore massimo e prosegue fino alla sua fine
  • sia H la sottolista di L che inizia dalla testa fino alla suo valore massimo escluso.

La lista L e` quindi uguale a H → M. Si vuole realizzare in C la funzione con prototipo:

node * head_is_max(node * head);

la quale modifichi la lista head in modo che risulti M → H. Se per esempio viene passata la seguente lista come argomento:

head → 1 → 4 → 10 → 5 → 7 → 9

allora, visto che il valore massimo e` 10, la chiamata head_is_max(head) restituisce il puntatore alla lista così modificata:

(valore di ritorno) → 10 → 5 → 7 → 9 → 1 → 4

ATTENZIONE: La funzione DEVE esclusivamente agire sui puntatori. NON deve quindi ne eliminare nodi, ne crearne di nuovi.

Il generico nodo della lista e` dichiarato del seguente tipo

typedef struct _node {
    int value;
    struct _node * next;
} node;

Feedback

Soluzione iterativa 
node * head_is_max(node * head) {
        node * curr, * prev;            /* Puntatori per scorrere lista */
        node * curr_max, * prev_max;    /* Puntatori per ricordare posizione max */

        curr = curr_max = head;
        prev = prev_max = NULL;
        while (curr != NULL) {
                if (curr->value > curr_max->value) {
                        /* Trovato valore piu` alto: aggiorno */
                        curr_max = curr;
                        prev_max = prev;
                }
                /* Scorro la lista */
                prev = curr;
                curr = curr->next;
        }
        if (curr_max != head) { /* Se non siamo in testa... */
                /* Richiudo lista su elemento di testa */
                prev->next = head;
                /* Chiudo la nuova lista dopo l'ultimo elemento */
                prev_max->next = NULL;
        }
        return curr_max;
}

Testo della domanda

Sia data la seguente definizione utilizzata per rappresentare delle stringhe di caratteri: 

struct string { 
   char *seq;
   int   n;
};

dove il campo seq è il puntatore al primo carattere di una sequenza (già allocata in memoria) di n caratteri.

Definire la funzione str_cat dal seguente prototipo:

struct string str_cat(struct string a_str[], const int length);

la quale:
1. ha due argomenti, rispettivamente:
   - a_str[]: un array di length elementi di tipo struct string;
   - length: il numero di elementi dell'array a_str[].
2. restituisce una variabile di tipo struct string che rappresenta la stringa concatenamento delle stringhe memorizzate in a_str[], secondo l'ordine crescente dell'indice.

Ad esempio se:

    a_str -> { {"Hello",5}, {" ",1}, {"World",5}, {"!",1} }

allora la chiamata str_cat(a_str, 4) restituisce una struct string così composta:

    {"Hello World!", 12}

Feedback

struct string str_cat(struct string a_str[], const int length){

    int i;              // variabile indice usata per scorrere l'array a_str
    int m;            // dimensione totale della stringa risultato della concatenazione
    char *p;          // puntatore usato per scorrere i caratteri della sequenza risultato
    struct string r; // struct string risultato (restituita per valore)
       
    for (m=0,i=0 ; i<length ; i++)
       m += a_str[i].n;
      
    r.seq = malloc(sizeof(char)*m);
    r.n = m;
    
    p = r.seq;
    struct string *e;
    struct string *end;
    for (e=a_str, end=a_str+length  ; e<end ; e++){
       char   *s = e->seq;
       int s_len = e->n;
       char *s_end = s+s_len;
       while (s<s_end )
          *p++ = *s++;
    }    
    return r;
}
