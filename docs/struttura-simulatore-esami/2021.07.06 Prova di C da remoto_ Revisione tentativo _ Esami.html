Sia data la seguente definizione di funzione su stringhe C:

typedef char * string;

/* scount(s,c): it counts how many c are in s */
int scount(string s, char c){
   int cnt=0;
   while (*s!='\0')
      cnt += ( *s++ == c ? 1 ; 0);
   return cnt;      
}
Data la stringa seguente:
string sentence = "The exam starts at 5.00 pm. The exam will be hosted on the Moodle platform remotely.";


Completare la chiamata alla funzione 'scount' affinché calcoli quanti caratteri 'm' sono nella seconda frase ("The exam will be hosted on the Moodle platform remotely.") della stringa sentence:

int n = scount( ,   );

Feedback

  • parte 1: sentence+27 (per iniziare a cercare da " The exam will be")
  • parte 2: 'm' (carattere da cercare con singoli apici)

Testo della domanda

[TEMPO STIMATO: 4 min]

Si consideri la seguente definizione di struttura C:

struct node { 
    int  val;
    struct node *next; 
};

Si completi la seguente funzione ricorsiva, la quale riceve un array v di n interi e inserisce in testa alla lista puntata da head, una struct node per ogni elemento di v. Il campo val dell'elemento della lista inserito in posizione i, deve valere v[i].

Per esempio, se vengono passati:

v = {4, 2, 6, 9};

head -> 8 -> 5

allora la lista ritornata dovra` essere

head -> 4 -> 2 -> 6 -> 9 -> 8 -> 5

struct node * ins_array_list(int v[], int n, struct node * head) {
        struct node * p;

        if (n==0)
                return head;
        p = malloc(sizeof(struct node));
        p->val = v[0];
        p->next = ins_array_list(,,);
        return p;
}

Feedback

  • parte 1: v+1 (vai alla prossima cella dell'array)
  • parte 2: n-1 (array di piu` corto di 1)
  • parte 3: head (la lista e` sempre la stessa)
