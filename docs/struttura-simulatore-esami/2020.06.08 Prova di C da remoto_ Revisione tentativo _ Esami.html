La seguente funzione C 'list_alloc' alloca  in memoria dinamica una lista di n elementi di tipo 'node', tipo  definito secondo la seguente struttura:

typedef struct _node { double *val;
                       struct _node *next;
                     } node;

Gli 'n' elementi della lista sono allocati in una struttura ad array.

La funzione restituisce il puntatore alla testa della lista così allocata, NULL nel caso l'allocazione non è potuta essere soddisfatta dal sistema operativo.

Completare il codice della funzione avendo l'accortezza di non inserire spazi bianchi non necessari.

 list_alloc(int n){
     int i;
     node *head=NULL;
     
     head = (node *)calloc(n, );  // alloca array di n nodi

     if (head!=NULL){
         node       *item     = head;
         const node *end_list = head+n-1;
         for ( item=head ; item < end_list ; item++)
             ;   // imposta il campo next a puntare al nodo successivo
         }
     }

     return head;
}

Risposta 1: node *

Risposta 2: sizeof(node)

Risposta 3: item->next=item+1
