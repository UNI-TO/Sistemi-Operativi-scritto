Testo della domanda

Sia data la seguente definizione di tipo utilizzata per rappresentare delle liste di valori interi: 

typedef struct _node {
    int val;
    struct _node * next;
} node;
Definire la funzione dal seguente prototipo:
node * list_trim(node * head);
la quale elimina e dealloca dalla lista head gli elementi seguenti con lo stesso valore val.
Se per esempio viene passata la seguente lista come argomento:

head → 1 → 1 → 3 → 3 → 3 → 2 → 7 → 4 → 4 → 4

allora la chiamata list_trim(head) restituisce il puntatore alla lista così modificata:

(valore di ritorno) → 1 → 3 → 2 → 7 → 4

ATTENZIONE: La funzione non deve allocare una nuova lista nell'ottenere il risultato.

Feedback

Soluzione iterativa 
node * list_trim(node * head)
{
	node * item; /* points to current node */
	node * p=l;   /* points to the predecessor of item */

	if (p != NULL && (item=p->next)!=NULL) {
		while (item!=NULL){
			if ( item->val == p->val ){
				p->next = item->next;
				free(item);
				item = p->next;
			} else {
				p = item;
				item=item->next;
			}
		}
	}
	return head;
}

Domanda 2

Risposta non data
Punteggio max.: 5

Testo della domanda

Data la seguente definizione di struttura C

struct string { 
	char *s;
	int length;
};

per la memorizzazione di stringhe e della loro lunghezza, si definisca una funzione C dal seguente prototipo:

struct string * str_array(char *str_cat, int size);
dove:

  • str_cat: rappresenta una concatenazione in memoria di stringhe C terminate dallo 0. Ad esempio, in
    str_cat-> "Unix System V\0is one of the first Unix\0operating systems\0"
    str_cat punta ad una concatenazione in memoria di 3 stringhe C così come rappresentate nel testo.
  • size: rappresenta il numero di stringhe nella concatenazione str_cat. Nell'esempio sopra, size ha valore 3.

La funzione restituisce un array di 'size' dati di tipo 'struct string' tale che ciascun suo elemento descriva, secondo l'ordine imposto dalla memorizzazione, la stringa corrispondente della concatenazione str_cat:

  • il campo 's' dell'elemento i-esimo dell'array punta alla i-esima stringa nella concatenazione;
  • il campo 'length' dell'elemento i-esimo dell'array è la lunghezza della i-esima stringa nella concatenazione.

Nell'esempio di cui sopra, la funzione restituirebbe un (puntatore ad un) array con

  1. il campo 's' del primo elemento che punta alla sotto-stringa "Unix System V"
  2. il campo 's' del secondo elemento che punta alla sotto-stringa "is one of the first Unix"
  3. il campo 's' del terzo (e ultimo) elemento che punta alla sotto-stringa "operating systems"
mentre i campi length sarebbero rispettivamente i valori: 13, 24, 17.

Feedback

Segue esempio di soluzione
struct string *str_array(char *str_cat, int size)
{
	struct string * sd;

	sd = (struct string *)calloc(size, sizeof(struct string));
	if (str_cat!=NULL){
		char *s=str_cat;
		int i;
		for (i=0;i<size;i++){
			sd[i].s = s;
			int length = 0;
		    while ( *s++ != '\0' )
			    length++;
			sd[i].length = length;
		}	
	}
	
	return sd;
}
